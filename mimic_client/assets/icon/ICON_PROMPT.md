# App Icon — source & regeneration

Custom exe/taskbar icon lives here. **Current `monitor_app/app.ico` is a PIL
placeholder** (brand-color `#863bff` rounded "G" tile) — generated by
`tools/gen_icon.py` because no designed source existed yet. Icon design is
deliberately deferred; this folder reserves the workflow.

## Step 1 — generate the artwork

Paste this into ChatGPT / DALL·E (English → better output):

```
A modern, minimal app icon for a real-time screen-capture tool that feeds
a computer-vision AI ("pixels in, actions out"). Concept: a rounded-square
monitor/viewport with screenshot crop-corner brackets at its four corners,
and at the center a stylized camera-lens / eye that subtly suggests machine
vision — a small neural spark or aperture ring inside the pupil. Flat vector
style, clean geometric shapes, crisp edges that stay legible at 16px. Primary
color a vivid purple (#863BFF) with a soft violet-to-blue gradient accent;
dark neutral base. Fully centered, single subject, transparent background,
no text, no letters, no drop shadows outside the icon silhouette. Delivered
as a 1024x1024 square, high detail, product-quality software icon.
```

Optional style modifiers (append one line):
- Minimal line: `use thin-line iconography, monochrome purple on transparent`
- Glassy depth:  `glossy glass-morphism, subtle depth and inner glow`
- Game flavor:   `replace the lens with a play-triangle merged into an eye`

## Step 2 — drop the file

Save the result here as **`icon_source.png`** — 1024×1024, transparent
background (PNG so PIL can consume it without svglib).

    monitor_app/assets/icon/icon_source.png

## Step 3 — regenerate app.ico + rebuild

Point `tools/gen_icon.py` at `assets/icon/icon_source.png` (instead of the
favicon.svg path), then:

    cd monitor_app
    python tools/gen_icon.py     # icon_source.png -> app.ico (multi-size)
    build.cmd                    # rc.exe embeds app.ico into the exe

The exe's icon updates on next prod build. Dev is unaffected.
